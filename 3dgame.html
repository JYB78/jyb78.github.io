3D Unity Game

A 3D game that I have created using the Unity and C# which I named "Rolling in the Deep". It is a balance game where the player will use their mouse to control the platform. The goal is to transport the ball into the end point box.

At each stage, the player will have at max 3 lives and will lose one whenever the ball falls off of the platform. The game contains a start menu, multiple game stages and a pause menu as shown in the images here.
